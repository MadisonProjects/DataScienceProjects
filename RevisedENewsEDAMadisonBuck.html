Define Problem Statement and Objectives¶

Context

The objective of an online news portal is to acquire new subscribers, and to achieve this, they analyze the actions taken by visitors to their website based on their interests. They plan to determine the effectiveness of a new feature by analyzing user responses to two variants of a product. This technique, known as a/b testing, helps companies decide which variant is more effective in attracting users based on a chosen metric. E-news Express has asked me to be their Data Scienctist, tasked with analyzing the effectiveness of a new landing page designed by the company's team and its weight in acquiring new subscribers. To achieve this, we randomly selected 100 users and divided them equally into two groups; the first group (control group) received the old landing page, while the second group (treatment group) received the new landing page. The data collected about the customers in both groups is stored in 'abtest.csv', and the task is to perform statistical analysis using this data to answer the following questions about the experiment.

Objectives

Do the users spend more time on the new landing page than the old landing page?

Is the conversion rate (the proportion of users who visit the landing page and get converted) for the new page greater than the conversion rate for the old page?

Does the conversion status depend on the preferred language?

Is the mean time spent on the new page same for the different language users?

A significance level of 0.05 will be used for all tests.

def histogram_boxplot(data, feature, xlabel, ylabel, figsize = (12, 7), kde = False, bins = None):
    """
    Boxplot and histogram combined

    data: dataframe
    feature: dataframe column
    figsize: size of figure (default (12,7))
    kde: whether to show the density curve (default False)
    bins: number of bins for histogram (default None)
    """
    f2, (ax_box2, ax_hist2) = plt.subplots(
        nrows=2, 
        sharex=True, 
        gridspec_kw={"height_ratios": (0.25, 0.75)},
        figsize=figsize,
    ) 
    
    sns.boxplot(
        data=data, x=feature, ax=ax_box2, showmeans=True, color="orange"
    )  
    
    sns.histplot(
        data=data, x=feature, kde=kde, ax=ax_hist2, bins=bins, palette="Set2"
    ) if bins else sns.histplot(
        data=data, x=feature, kde=kde, ax=ax_hist2
    )  
    
    ax_hist2.axvline(
        data[feature].mean(), color="green", linestyle="--"
    )  
    
    ax_hist2.axvline(
        data[feature].median(), color="black", linestyle="-"
    )  
    
    ax_box2.set_xlabel("", fontsize = 16)
    ax_hist2.set_xlabel(xlabel, fontsize = 16)
    ax_hist2.set_ylabel(ylabel, fontsize = 16);

def labeled_barplot(data, feature, xlabel, ylabel, perc = False, n = None):
    """
    Barplot with percentage at the top

    data: dataframe
    feature: dataframe column
    perc: whether to display percentages instead of count (default is False)
    n: displays the top n category levels (default is None, i.e., display all levels)
    """

    total = len(data[feature]) 
    count = data[feature].nunique()
    if n is None:
        plt.figure(figsize=(count + 1, 5))
    else:
        plt.figure(figsize=(n + 1, 5))

    plt.xticks(fontsize=15)
    
    ax = sns.countplot(data=data, x=feature, palette="Set2", order=data[feature].value_counts().index[:n].sort_values())

    for p in ax.patches:
        if perc == True:
            label = "{:.1f}%".format(100 * p.get_height() / total) 
        else:
            label = p.get_height()

        x = p.get_x() + p.get_width() / 2 
        y = p.get_height()    

        ax.annotate(label, (x, y), ha="center", va="center", size=12, xytext=(0, 5), 
                    textcoords="offset points")  
    
    ax.set_xlabel(xlabel, fontsize = 16) 
    ax.set_ylabel(ylabel, fontsize = 16) 
    
    plt.show()

Step 1: Define the null and alternate hypotheses¶

null hypothesis:

𝐻0: 𝜇1=𝜇2

alternative hypothesis:

Ha: 𝜇1<𝜇2

𝜇1: mean time spent on old landing page

𝜇2: mean time spent on new landing page

Step 2: Select Appropriate test¶

Comparing two means (old and new) with STD unknown would require a 2-sample independent t-test.

The time data is continuous and the sample is random. This should result in fairly normal distribution.

Step 3: Decide the significance level¶

The project expressed we will use a .05 significance level (95%)

Step 4: Collect and prepare data¶

Step 7: Draw inference¶

We can reject the null hypothesis as the p-value is less than that of the 0.05 level of significance. (0.000139 < 0.05) This means there is enough evidence at a 5% significance level to determine that the mean time spent by the user population directed to the new page is larger than that for the user population which used the old site.
